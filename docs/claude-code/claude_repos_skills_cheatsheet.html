運用・コスト・安全

OPS

チーム導入で効く実務系。コスト可視化とセキュリティレビューはガバナンスに直結。

ccusage/ccusagecost
~/.claude/projects/*.jsonl を読み、トークン・コストを可視化。オフライン・APIコール無しで使えるものが多い。
anthropics/claude-code-security-review★ ~5.2k
変更差分を Claude でセキュリティ分析する GitHub Action。PR ゲートに組み込める。ブランチ保護運用と相性が良い。
導入前チェック（ガバナンス）：
  • スキル/プラグインは任意コードが走り得る。Anthropic も「信頼できる元のみ・導入前にレビュー」を推奨。
  • ★や採用数はスナップショット。リンク先で現況・最終更新・ライセンスを必ず確認。
  • 一度に大量導入しない。カテゴリごとに1つから。衝突と環境汚染の元になる。
補足（要確認）：2026年4月の方針変更で、一部サブスク（Pro/Max）からサードパーティのエージェントフレームワーク利用が制限されたという報道がある。公式/コミュニティの線引きは変わり得るので、最新の規約を確認のこと。
